# Brazilian-E-Commerce-Dataset-ETL-Visualization
Repositório destinado ao trabalho da disciplina de Sistema de Banco de Dados 2 (SBD2), do curso de Engenharia de Software da UnB, orientado pelo professor Thiago Luiz. O projeto consiste no desenvolvimento de um pipeline de ETL utilizando a Arquitetura de Medalhão (Bronze, Silver e Gold) para análise de dados do E-commerce Brasileiro (Olist).
